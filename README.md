# VR Portfolio Scripts

VR 강아지 키우기 프로젝트에서 제가 담당한 스크립트를 정리한 레포입니다

##담당 기능
-이름 입력 키보드
-씬 전환 및 로딩(Loading Scene->Target Scene)
-문 트리거 상호작용(BathRoom,Walk씬으로 이동시)

##폴더 구조
-"Scripts/SceneFlow":씬 전환/로딩 관련
-"Scripts/Door":문 상호작용 관련
-"Scripts/UI":이름 입력/UI Raycast관련

##주요 스크립트
-"DogName":강아지 이름 입력 및 저장
-"HouseLoading","WalkLoading","BathroomLoading":로딩 씬 후 전환 흐름
-"DoorOpen":문 트리거 기반 이동 처리

##사용 기술
-Unity,C#
